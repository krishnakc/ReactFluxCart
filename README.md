# ReactFluxCart
This application is built with NodeJS, React, Flux, Gulp

With node installed, please install **gulp** globally:

`npm i -g gulp`

Install local dependecies

`npm install`

Now run `gulp` in the project folder. This builds the project in the `dist` folder and watches for any changes. We can serve via httpster is a great option for this (`npm i -g httpster`)
